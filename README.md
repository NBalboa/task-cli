# How to use

## Option 1

**_npm run start `cmds`_**

## Option 2

**_npm link_** <br>
task-cli `cmds`

cmds are:

<ol>
  <li>add "task description"</li>
  <li>update *id* "new description"</li>
  <li>delete *id* </li>
  <li>mark-in-progress *id* </li>
  <li>mark-done *id* </li>
  <li>list</li>
  <li>list *status*</li> //status are "todo, in-progress", "done"
</ol>
